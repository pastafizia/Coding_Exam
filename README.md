# Coding Exam Repository

This repository contains the material developed for my Coding exam.
It includes the solved exercises for the first module, the second module, and the related final project.

## Repository Structure

* Module 1 – Solved exercises from the first module
* Module 2 – Extension of a pre-existing exercise by introducing new classes organized through class inheritance, applying the Object-Oriented Programming (OOP) concepts studied during the course
* Project – Final project developed as part of the exam

## Purpose

The purpose of this repository is to collect and organize the work completed during the course, including exercises and the final project.

## Notes

All materials in this repository are intended for academic purposes.
